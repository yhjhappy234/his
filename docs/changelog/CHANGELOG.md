# 版本变更记录 (CHANGELOG)

> 本文档记录 HIS 医院信息系统的版本发布历史。

## [1.2.0] - 2026-04-06

### 新增功能

#### 系统架构
- 新增双模式部署支持（微服务模式/单体模式）
- 新增 SQLite 数据库支持，实现轻量化部署
- 新增 Windows 平台原生支持

#### 系统管理模块
- 用户管理：用户注册、审核、信息维护
- 角色权限：基于 RBAC 的权限管理
- 数据字典：全局字典管理
- 系统参数：系统配置管理
- 操作日志：用户操作记录
- 审计日志：安全审计功能

#### 门诊管理模块
- 预约挂号：支持线上预约、现场挂号
- 排班管理：医生排班配置
- 分诊排队：智能分诊、排队叫号
- 医生工作站：病历书写、处方开立
- 收费结算：门诊收费、医保结算

#### 住院管理模块
- 入院登记：入院申请、床位分配
- 床位管理：床位状态管理
- 医嘱管理：长期/临时医嘱管理
- 护理管理：护理记录、生命体征
- 出院管理：出院结算、出院小结

#### 药房管理模块
- 药品管理：药品目录维护
- 库存管理：库存查询、效期预警
- 发药管理：处方审核、发药确认
- 采购管理：采购订单、入库验收

#### 语音呼叫模块
- 诊室叫号播报
- 药房取药通知
- 检验报告通知
- 多音频设备支持

### 技术特性

- 采用 Spring Boot 3.2.x + JDK 21
- 支持 Windows 服务模式运行
- 支持 Windows SAPI 语音引擎
- 支持 Web 展示终端（大屏、自助机）
- 支持 JWT Token 认证
- 支持 Swagger API 文档

## 版本命名规范

版本号格式: `主版本号.次版本号.修订号`

- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向下兼容的功能新增
- **修订号**: 向下兼容的问题修复

---

## 变更类型

- `新增` (Added): 新功能
- `变更` (Changed): 现有功能的变更
- `弃用` (Deprecated): 即将移除的功能
- `移除` (Removed): 已移除的功能
- `修复` (Fixed): 问题修复
- `安全` (Security): 安全相关修复
